# Dink Card

Dink Card is a digital card service platform that allows verified users to add funds, request virtual cards, manage their card activity, and contact support when needed.

The platform is built to provide a simple, secure, and user-friendly card experience for customers.

## Main Features

- User account registration and login
- Customer verification before card access
- Add money to account balance
- Request virtual cards
- Manage card information
- Submit support requests
- Admin management panel
- Secure payment and card processing
- Terms and policy acceptance before using the service
